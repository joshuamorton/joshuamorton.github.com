I do projects

ClassRank

This is a webapp I'm developing to use a collaborative filtering based approach to provide more useful metrics when students attempt to register for classes. This was inspired by the often asked question among new students at Georgia Tech, "should I take CS1332 and CS2110 in the same semester". It is written in python and runs on Tornado.

Relevant highlights include the document in which I outline the collaborative filter I plan to use and the classes used to implement databases with varying schema.

This project is a huge pet of mine, and something I actually want to develop into a working, useful tool.

Csp-site

Another webapp, developed in Django with The Agency at Georgia Tech. I created the web side of a tool that would, given inputs of constraints regarding classes and schedules, create and provide a course schedule that met those constraints. The tool no longer works and was never "stable", however it is something I would like to continue development on at some point.

Snippets

I keep a repo of miscellaneous code snippets on github, among them are a few of interest.

A brainfuck interpreter written in python was what I chose to write for my final project in the intro to programming course at Georgia Tech, we could chose anything as long as it demonstrated an understanding of concepts.

A basic lisp interpreter, fun parts are that parsing the code is done in a single regex statement, and that you are able to import arbitrary python modules and therefore gain access to the standard library. Concievably, you could write a basic webapp in just a few lines, within the lisp. This was written with Nick Johnson when we stumbled upon the stupid idea of writing a lisp parser in regex.

Obiwan Extras is an extension to the Obiwan module, a python type-assurance module that uses a gorgeous syntax. It was written as a proof of concept with regards to the August 2014 proposal by Guido van Rossum to use MyPy syntax as python's officially supported typing syntax. It is my opinion that the syntax used by Obiwan is superior, and since Obiwan lacked the ability to create type-assured compound types, I implemented a simple tool that could provide such objects.

Overloading

On the subject of static types in python, well before I knew about annotations, and before the propsoal to officially support types was introduced, I wrote a module to provide function (method) overloading, and a convoluted and weak form of pattern matching to python. Its currently not very efficient, it does some very, very stupid things, which the readme outlines, but could be improved, although not to any useful end.
